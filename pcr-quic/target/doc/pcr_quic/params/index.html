PCR-QUIC Transport Parameter

Defines the pcr_quic transport parameter for negotiating PCR mode between QUIC endpoints.

§Wire Format

PCR-QUIC Transport Parameter {
    Version (8),
    KEM ID (16),
    Rekey Interval (32),    // seconds
    Window W (32),          // max out-of-order packets
}

§Negotiation

PCR mode is enabled only when both peers advertise the pcr_quic transport parameter with compatible versions and KEM IDs.
