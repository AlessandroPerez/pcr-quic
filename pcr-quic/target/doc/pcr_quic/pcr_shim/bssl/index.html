Low-level FFI bindings to BoringSSL via the C crypto shim PCR-QUIC BoringSSL FFI bindings

Low-level FFI declarations for the PCR-QUIC crypto shim. These functions are implemented in C (crypto_shim.c) and link against BoringSSL.
